# SIGIL bootstrap

This vault is the memory and mind of a SIGIL agent. Edit notes, link them
with `[[wikilinks]]`, and the agent's context changes — no retraining.

Begin with `intent.md` (the autonomy contract) and `persona.md` (who the
agent is). Use `[[grave]]` to tombstone a note (stops auto-surfacing).
Stamp beliefs with `claim:` so the conflict protocol can protect them.
